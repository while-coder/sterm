# STerm

STerm 是一个跨平台 SSH / SFTP 终端客户端，支持 Windows、macOS、Linux、Android 和 iOS。

- [下载 STerm](https://github.com/while-coder/sterm/releases)
- [使用文档](https://while-coder.github.io/sterm/)
- [版本记录](https://github.com/while-coder/sterm/releases)

## 主要功能

- SSH 多会话终端、分屏、广播输入、跳板机、代理与端口转发
- SFTP 文件浏览、上传、下载、断点续传和代码查看
- AI 终端助手，支持命令执行确认和多种模型接口
- 主密码保护、加密导入导出与多种配置同步方式
- 桌面端应用内检查、下载和安装更新

详细使用方式请查看[完整文档](https://while-coder.github.io/sterm/)。
