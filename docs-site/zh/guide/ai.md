# AI 终端助手

AI 面板属于当前 SSH 会话。它可以回答问题、读取连接建立时缓存的远端环境信息，也可以在你确认后直接在当前交互式 shell 中执行命令。

## 模型服务

在「设置 → AI」中填写协议、接口地址、模型和 API Key。STerm 支持：

- OpenAI-compatible Chat Completions
- OpenAI Responses API
- Anthropic Messages API
- Ollama

API Key 通过操作系统凭证库保存，不会写进普通设置文件。

## 附加终端现场

发送问题前可以选择：

- **选区：** 附加当前终端里选中的文字。
- **最近输出：** 附加当前会话保留的最近一段终端输出。

现场只附加到这一次请求。终端内容会被当作待分析的数据，而不是对模型的新指令。

## 命令执行

模型请求运行命令时，STerm 会展示用途说明和完整命令，由你选择执行或拒绝。设置里可以开启自动批准，但生产服务器更适合保留人工确认。

命令运行在真实交互式会话中，因此 `cd` 等操作也会改变终端以及 SFTP 跟随功能看到的当前目录。

AI 请求可以随时中止，已经完成的工具结果仍会保留在对话中，方便核对答案依据。
