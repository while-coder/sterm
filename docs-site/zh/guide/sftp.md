# SFTP 文件管理

SFTP 面板直接复用当前 SSH 连接，不需要再次登录。

## 浏览和管理

通过工具栏、右键菜单或长按可以：

- 浏览目录并返回 Home
- 创建文件和文件夹
- 重命名、单项删除或批量删除
- 修改 `644`、`755` 等 Unix 权限
- 终端切换目录后自动跟随到相同位置

## 上传与下载

支持单个文件、多选项目和完整文件夹。把本地文件或目录拖入 SFTP 面板时，目标位置会固定为入队那一刻打开的远端目录。

所有传输进入统一队列，并按照设置中的并行上限执行。传输面板会显示进度，也可以只查看当前会话的任务。

## 断点续传与重试

开启「断点续传」后，取消或断线时会保留未完成文件。重新发起或点击重试时，只要源文件大小仍然匹配，就会从已有字节继续。

文件夹续传会跳过已经完整传输的文件。如果你明确要用一个同样大小但内容不同的文件覆盖旧文件，请临时关闭断点续传。

## 查看和编辑文本文件

较小的文本文件会在内置查看器中打开，并为 JSON、Markdown、Python、YAML、JavaScript、TypeScript 等常见格式提供语法高亮。保存后内容会写回远端。较大的文件或二进制文件可以先下载，再用系统默认程序打开。
