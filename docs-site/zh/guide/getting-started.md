# 快速开始

STerm 是一个跨平台 SSH / SFTP 客户端，把远程终端、文件管理和 AI 排查放在同一个工作区里。每个连接都有独立的终端、SFTP 和 AI 面板，多台机器可以通过标签页或分屏同时保持在线。

## 安装

前往 [GitHub Releases](https://github.com/while-coder/sterm/releases) 下载最新版本。

- **Windows：** NSIS 安装包
- **macOS：** DMG 或 app 包
- **Linux：** AppImage 或 DEB
- **Android：** Release 附件中的 APK

桌面端可在「关于」页面检查更新，有新版本时也会主动提示。

## 第一次连接

1. 首次启动时设置主密码，STerm 会用它加密本地机器列表。
2. 选择「新建连接」。
3. 填写显示名称、主机、端口、用户名和认证方式。
4. 选择「保存并连接」。
5. 首次连接会显示服务器主机指纹。请先与管理员提供的指纹核对，再保存信任。

只有勾选「记住密码 / 口令」后，连接密码、私钥口令或代理密码才会保留。

## 工作区

每个标签页都有自己的终端、SFTP、AI 对话和面板布局。顶部可打开双会话分屏，也可以把键盘输入广播到所有已连接标签。广播开启前一定会再次确认。

STerm 可以在下次启动时恢复上次打开的已保存连接。没有保存密码的密码连接会被跳过，避免后台弹出一批无法完成的登录。
